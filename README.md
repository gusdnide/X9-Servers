# X9 Servers
Sistema web para anunciar servidores de jogos usando PDO PHP, Mysql, Api’s do google e HTML\CSS com painel de controle e cargos para a administração do site.

## Linguagens Usada
* PHP
* MYSQL
* JS
* HTML\CSS

## Auxilios
* Bootstrap
* Font Awesome
